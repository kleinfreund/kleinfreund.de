  • October 2018 – March 2019 (approximately)

    Started writing my bachelor thesis. I will be working with args.me, an argument search engine built by Webis.

    As part of the thesis, the args API will be extended so that one can retrieve a list of counter-arguments for a specific argument. Additionally, the args.me website will receive a fully responsive layout. Finally, this new layout will be evaluated in a user study.

  • April 2018 – March 2019

    Developing Disraptor for the Webis network at Bauhaus-Universität Weimar.

    Disraptor tries to decouple the development of an arbitrary web application from the need to implement auxiliary functions like authentication, user management and social networking. The plan is to render the web application inside a Discourse instance and to use the named features that come with it.
